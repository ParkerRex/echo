# Project Brief

## Objective

Build a fully automated pipeline for uploading `.mp4` video files to YouTube, enriched with AI-generated transcripts, titles, chapters, and descriptions. The system leverages Google Cloud Run, Vertex AI (Gemini), and a modern React frontend to streamline video publishing and metadata generation.

## Scope

- Automate ingestion, processing, and publishing of video content.
- Use AI to generate high-quality, consistent metadata for YouTube uploads.
- Provide a real-time, user-friendly frontend for upload, monitoring, and content management.
- Ensure scalable, cloud-native architecture with robust local development via Docker.
- Support both daily and main channel content workflows.
- Integrate comprehensive testing, monitoring, and CI/CD for reliability and maintainability.
